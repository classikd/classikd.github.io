# Xonfluence Neo

Xonfluence Neo is a MOD of Xonfluence with a more modern interface, more options and better use of the main color.

![alt text](https://github.com/classikd/skin.xonfluence.neo/blob/master/resources/icon.png?raw=true)

**Note**: The skin uses a new ID, and therefore the skin does not overwrite Xonfluence


## Installation

This skin can be installed either by downloading and installing the ZIP file manually, or by adding and using my repository for automatic updates.

#### ZIP Install:
[https://github.com/classikd/skin.xonfluence.neo/releases/latest](https://github.com/classikd/skin.xonfluence.neo/releases/latest)

#### Repository Install:
[https://classikd.github.io/](https://classikd.github.io/)
